# Sales Performance Analysis

## Objective
To analyze retail sales data and identify trends, top-performing products, 
profitability issues, and regional performance using Python.

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib
- Jupyter Notebook

## Key Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Category-wise, region-wise, and product-wise analysis
- Monthly sales and profit trend analysis

## Key Insights
- Identified top-performing and loss-making product categories
- Discovered seasonal sales patterns
- Provided data-driven recommendations for improving profitability
